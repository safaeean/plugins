راحتی کار

سازگاری با ووکامرس

این افزونه با نسحه های مختلف ووکامرس سازگار است و تست شده است.

کار با فایل اکسل معمولی

برای کار با این افزونه نیازی به داشتن فایل اکسل با فرمت خاصی وجود ندارد. هر فایل اکسلی که حداقل یک ستون مربوط به شناسه محصول داشته باشد کافیست

نصب راحت

فقط با نصب افزونه منویی به پنل مدیریت وردپرس شما اضافه میگردد و نیازی به کانفیگ خاصی برای این افزونه وجود ندارد فقط کافیست شناسه های محصول را وارد کنید.

نامحدود
